!!! DO THIS AT HOME!!!

  1. Go to github.com/idiotidoit/Card-stack-games
  2. Click Fork in the top right (Make sure you are signed into github for this)
  3. Log off and go to netlify.app
  4. Sign up using github
  5. Once in, Click “Add new site”  and click “Import an existing project”
  6. Click github and sign in with the same login you used for github earlier
  7. Find yourname/Card-stack-games And click on it
  8. Scroll down and click deploy site
  9. Done!

CHANGING SITE DOMAIN

  1. Go to “Site settings” On your netlify.app, app
  2. Click change site name
  3. Change it to something with actual words that you will remember
  4. Your website will now be hosted at (Site name).netlify.app

FOR NON NETLIFY.APP DOMAINS

  1. Go to Freenom.com or any domain / Dns host (I use freenom for the free domains)
  2. Get your domain
  3. Go back to your website and go to domain settings
  4. Hit Add custom domain
  5. Type in your full domain (without https://www. , It is automatically added)
  6. Change the dns or nameserver settings to the netlify settings for netlify dns
  7. Done

UPDATING SITE

  1. Open github.com
  2. Go to your profile in the top left (The small circle pfp with the dropdown)
  3. Open it and click Your repositories
  4. Find “Card-Stack-Games”
  5. Open it and find the “Sync fork” option under code

!! IF IT DOES NOT HAVE SYNC FORK THEN THAT MEANS THERE ARE NO NEW UPDATES!!

  1. Click it and click the Green button
  2. Done
