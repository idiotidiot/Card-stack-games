!!! DO THIS AT HOME!!! Go to github.com/idiotidoit/Card-stack-games Click Fork in the top right (Make sure you are signed into github for this) Log off and go to netlify.app Sign up using github Once in, Click “Add new site” and click “Import an existing project” Click github and sign in with the same login you used for github earlier Find yourname/Card-stack-games And click on it Scroll down and click deploy site Done! CHANGING SITE DOMAIN Go to “Site settings” On your netlify.app, app Click change site name Change it to something with actual words that you will remember Your website will now be hosted at (Site name).netlify.app FOR NON NETLIFY.APP DOMAINS Go to Freenom.com or any domain / Dns host (I use freenom for the free domains) Get your domain Go back to your website and go to domain settings Hit Add custom domain Type in your full domain (without https://www. , It is automatically added) Change the dns or nameserver settings to the netlify settings for netlify dns Done UPDATING SITE Open github.com Go to your profile in the top left (The small circle pfp with the dropdown) Open it and click Your repositories Find “Card-Stack-Games” Open it and find the “Sync fork” option under code !! IF IT DOES NOT HAVE SYNC FORK THEN THAT MEANS THERE ARE NO NEW UPDATES!! Click it and click the Green button Done
